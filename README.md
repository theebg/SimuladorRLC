# Simulador Interactivo de Circuito RLC (UNAD)

Este proyecto es un simulador web interactivo diseñado para visualizar el comportamiento de la carga, la intensidad de corriente y el voltaje en el inductor de un circuito RLC en serie.

![Banner del proyecto](logo-unad.png)

## 🚀 Características
- **Simulación en tiempo real**: Visualiza la evolución de las constantes físicas mediante visores digitales y un osciloscopio integrado.
- **Control de Velocidad**: Multiplicadores de tiempo ($\times 0.01$ a $\times 3$) para observar fenómenos rápidos paso a paso.
- **Notación Matemática**: Ecuación diferencial renderizada con **KaTeX** basada en los parámetros ingresados.
- **Identidad Institucional**: Diseño alineado con la paleta de colores y logos de la UNAD.
- **Totalmente Web**: No requiere instalación, funciona directamente en el navegador.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6)
- [Chart.js](https://www.chartjs.org/) para los gráficos.
- [KaTeX](https://katex.org/) para las fórmulas matemáticas.

## 📦 Instalación y Uso
1. Descarga o clona este repositorio.
2. Abre el archivo `index.html` en cualquier navegador moderno.
3. Ingresa los parámetros $L$, $R$, $C$, $E$ y las condiciones iniciales.
4. Haz clic en **▶️ SIMULAR**.
